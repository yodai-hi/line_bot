# LineBot with Google Services
## 環境
```
macOS Catalina 10.15.4
npm 6.14.5
node v14.2.0
clasp 2.3.0
```

## 導入方法

``` shell script
$ mv ./assets/.env.exmple ./assets/.env
$ mv ./.clasp.json.example ./.clasp.jon
```
.envと.clasp.jsonの中を書き換える
